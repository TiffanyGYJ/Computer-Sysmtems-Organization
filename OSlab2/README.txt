Input is fileName.txt.
So please store all of the input inside txt file and save the file name, put them inside src folder.
Then open the terminal
Use command cd to go to the src file
Then type javac Scheduler.java
it will create javac file for Scheduler.java
then run the program by typing java Scheduler inputFileName.txt 
			    OR java Scheduler --verbose inputFileName.txt
inputFileName is the name of the input file you would like to run.
